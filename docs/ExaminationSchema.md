
# 문제 파일 변환 형식
- 프로그래밍적으로 처리 가능하도록, 원본 문제지를 아래 형식으로 관리한다.

```yaml
종목명: ---
회차: ---
1교시:
  1번: ---
  2번: ---
  ...
2교시:
  1번:
    대문제: ---
    소문제:
      - ---
  ...
3교시:
  1번:
    대문제: ---
    소문제:
      - ---
  ...
4교시:
  1번:
    대문제: ---
    소문제:
      - ---
  ...
      
```


# 문제 구분을 위한 ID
- 하나의 문제에 대한 ID는 다음과 같이 정의한다.
```
{종목명}.{회차}.{교시}.{문제번호}
```
